# Indian Bike Sales Analysis Dashboard

## Project Overview

This project analyzes Indian bike sales data using Python, SQL, and Power BI. The goal is to uncover business insights related to revenue, sales performance, customer demographics, payment methods, and regional trends.

The project demonstrates the complete Data Analytics workflow, including data cleaning, exploratory data analysis (EDA), SQL querying, and interactive dashboard creation.

---

## Dataset Information

The dataset contains 15,000 bike sales records with the following attributes:

* Sale ID
* Sale Date
* City
* State
* Brand
* Model
* Customer Age
* Gender
* Payment Mode
* Ex-Showroom Price
* Discount
* Final Price
* Quantity Sold
* Revenue

The dataset also includes missing values to simulate real-world business data.

## Key Business Insights

* Premium bike brands generated higher revenue.
* Commuter bike brands contributed the highest sales volume.
* Revenue varied significantly across cities and states.
* Loan and digital payments were among the most preferred payment methods.
* The 26–35 age group represented a significant customer segment.
* Seasonal trends affected bike sales performance.
